API Colombia 🇨🇴

Proyecto realizado en Python utilizando la API pública de Colombia para consultar información del país, departamentos, regiones y atracciones turísticas.

📌 Descripción

Este proyecto consume la API de Colombia:

API Colombia

Permite visualizar:

Información general de Colombia
Lista de departamentos y sus capitales
Regiones y descripciones
Atracciones turísticas
Menú interactivo para seleccionar consultas

🛠️ Tecnologías utilizadas
Python 3
Requests
API REST

📦 Instalación
1. Clonar el repositorio
git clone https://github.com/usuario/nombre-repositorio.git

2. Entrar al proyecto
cd nombre-repositorio

3. Crear entorno virtual (Opcional)
python -m venv venv

4. Activar entorno virtual
Windows
venv\Scripts\activate
Linux / Mac
source venv/bin/activate

5. Instalar dependencias
pip install requests
▶️ Ejecución

Ejecutar el archivo principal:

python main.py
